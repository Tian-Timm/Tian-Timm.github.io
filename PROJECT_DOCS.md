# Personal Portfolio Project Documentation

## Project Overview
This is a personal portfolio website built with modern web technologies. It showcases your profile, skills, projects, and provides a way for others to contact you. The design focuses on minimalism, interactivity, and responsiveness.

## Tech Stack
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (based on Radix UI)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Project Structure & Key Components

### 1. Main Application Logic (`/app`)
- **`app/page.tsx`**: The main entry point of the website. It acts as a layout container that assembles all the individual sections (Hero, About, TechStack, etc.) in a vertical scroll layout.
- **`app/layout.tsx`**: Defines the global HTML structure, fonts, and theme providers.
- **`app/globals.css`**: Global CSS styles and Tailwind directives.

### 2. UI Components (`/components`)
All visual elements are modularized in the `components` directory.

#### Core Sections (in order of appearance):
- **[Navbar](components/navbar.tsx)**: The top navigation bar. Handles scrolling to different sections and mobile menu toggle.
- **[Hero](components/hero.tsx)**: The landing section. Contains your introduction, typing effect, social links, and your profile photo.
- **[About](components/about.tsx)**: A brief "About Me" section describing your philosophy and background.
- **[TechStack](components/tech-stack.tsx)**: "My Creative Toolkit". Displays your skills and tools using a card-based layout with Lucide icons.
- **[Projects](components/projects.tsx)**: Showcases your selected works. Each project card includes a description, tags, and links.
- **[Testimonials](components/testimonials.tsx)**: Displays endorsements or reviews from others.
- **[Contact](components/contact.tsx)**: A contact form and list of social links/contact info.
- **[Footer](components/footer.tsx)**: The bottom section with copyright info.

#### Shared UI Elements:
- **`components/ui/`**: Contains reusable low-level components (Buttons, Cards, Inputs, Badges, etc.) generated by Shadcn UI.
- **`components/icons.tsx`**: Custom icon definitions (e.g., the Twitter/X logo).
- **`components/theme-provider.tsx`**: Manages light/dark mode theming.

### 3. Static Assets (`/public`)
- **`public/profile.jpg`**: Your profile picture displayed in the Hero section.
- **`public/`**: Other static files like favicons and placeholder images.

## Configuration Files
- **`tailwind.config.js`**: Customization for Tailwind CSS (colors, animations, breakpoints).
- **`tsconfig.json`**: TypeScript compiler options.
- **`package.json`**: Project dependencies and scripts.

## How to Start the Project

### Prerequisites
- [Node.js](https://nodejs.org/) (Version 18 or higher recommended)
- A package manager like `npm`, `pnpm`, or `yarn`.

### 1. Install Dependencies
Open your terminal in the project directory (`d:\Vic\mycode\myportfolio`) and run:
```bash
npm install
# or
pnpm install
# or
yarn install
```

### 2. Run Development Server
To start the local development server with hot-reloading:
```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

### 3. Build for Production
To create an optimized production build:
```bash
npm run build
# or
pnpm build
```

### 4. Start Production Server
To preview the production build locally:
```bash
npm start
# or
pnpm start
```
